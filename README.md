
# Visa Application System (MERN Stack)

This is a simple MERN (MongoDB, Express, React, Node.js) stack application for handling visa applications.

## Features
- Users can apply for visas
- users can view the Details table

## How to Run

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm start
```
